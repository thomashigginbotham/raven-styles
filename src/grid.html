Grid System

A flexible grid system for making diverse layouts is a necessity for any CSS framework, and Raven doesn’t disappoint. Creating grids has never been easier!

Minimal HTML

Basic Columns Example

To display child elements side by side as columns, add a data-columns attribute. It couldn’t be simpler!

Column
<div data-columns>
  <div>Column</div>
  <div>Column</div>
  <div>Column</div>
</div>

What’s Happening?

When the data-columns attribute is present, the direct child elements are displayed inline in a row.

By default, each column is only as wide as its contents. However, you can stretch the columns equally to fill out the row by adding the data-fill attribute to each column element or by setting the data-columns value to "fill".

Consistent Spacing

Row and Column Gaps

Layouts Based on Container Width

Responsive Changes

The same column configuration for all screen sizes won’t cut it. Responsive design is nothing new and is supported by any major CSS framework. However, we’ve gone a step further.

Our columns are responsive to their parent container, not the viewport. That’s right. We have container queries! Resize the example below to try it yourself.

Column
Column
Column
<div data-rows data-columns-sm data-gap="md">
  <div data-width-sm="quarter">Item</div>
  <div data-width-sm="half">Item</div>
  <div data-width-sm="quarter">Item</div>
</div>

What’s Happening?

The data-rows attribute means the three child elements display as rows by default. However, when the width of the row reaches the "small" range, the data-columns-sm attribute tells the row to display its children as columns.

Each of the child elements has a data-width-sm attribute that tells it what width it should be when its parent reaches the "small" width.
